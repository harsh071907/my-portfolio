# Prashant Kumar - Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing projects, skills, and experience.

## Features

- ✨ **Modern UI**: Clean and professional design
- 📱 **Responsive**: Works seamlessly on all devices
- ⚡ **Fast**: Built with Vite for optimal performance
- 🎨 **Interactive**: Smooth animations and transitions
- 📊 **Comprehensive Sections**:
  - Hero Section
  - About Me
  - Skills & Technologies
  - Featured Projects
  - Professional Experience
  - Contact Form
  - Social Links

## Tech Stack

- **Frontend Framework**: React 18.2.0
- **Build Tool**: Vite 5.0.8
- **Styling**: CSS3 with modern features
- **Icons**: React Icons 5.0.1
- **Utilities**: JavaScript (ES Module)

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/harsh071907/my-portfolio.git
cd my-portfolio
```

2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build:
```bash
npm run preview
```

## Customization

### Personal Information

Update the following files with your information:

1. **index.html**: Update the `<title>` tag
2. **Hero.jsx**: Update name, subtitle, and description
3. **About.jsx**: Add your about information and stats
4. **Skills.jsx**: Update your technical skills
5. **Projects.jsx**: Add your projects
6. **Experience.jsx**: Add your work experience
7. **Contact.jsx**: Update contact information
8. **Footer.jsx**: Update social media links

### Styling

- Colors are defined in `index.css` using CSS variables
- Modify the `:root` variables to change the color scheme
- Each component has its own CSS file for modular styling

### Social Links

Update social media links in:
- `Hero.jsx`
- `Footer.jsx`

Replace placeholder URLs with your actual social media profiles.
